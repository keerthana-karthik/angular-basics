Key Snippets

    To create template driven form add (ngSubmit) and create refetence :
        <form (ngSubmit)="onNgSubmit()" #f="ngForm">...</form>

    To add a control part of NgForm add ngModel and name attributes :
        <input type="checkbox" ngModel name="chekbx" />

    To reference a control and conditionally render another section
        <select ngModel name="selectbox" #selectbox="ngModel">...</select>
        *ngIf="selectbox.control.touched"
    
    To get access to template and DOM before any method is triggered
        @ViewChild("f") templateDrivenForm: NgForm;

    Setting Default
        templateDrivenForm.setValue(KEY VALUE PAIR AS OBJECT);
    
    Controlling Navigation with canDeactivate
        canDeactivate: [CanFormDeactivateGuard]
        To use CanFormDeactivateGuard guard for a component
            create canDeactivate() method within the component.
