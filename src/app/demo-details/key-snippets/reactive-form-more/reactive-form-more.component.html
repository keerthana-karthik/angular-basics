Key Snippets

    To create template driven form add (ngSubmit) and create refetence :
        <form (ngSubmit)="onNgSubmit()" [formGroup]="reactiveForm">...</form>
        In ts file "this.reactiveForm = new FormGroup("

    To add a control part of NgForm add ngModel and name attributes :
        <input type="checkbox" formControlName="chekbx"" />

    To reference a control and conditionally render another section
        In ts file "selectbox: new FormControl(null)"
        *ngIf="reactiveForm.get('selectbox').touched"
