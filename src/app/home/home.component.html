Reactive Form

  • Uses ReactiveFormsModule
  • added to elementss
  • formControlName attribute added to elements
  • (ngSubmit)="onSubmit()" [formGroup]="signForm"

Component Interaction

  • Binding Data from Parent to child
  • @Input('aliasName') in ts, [aliasName]=value in html
  • Emit event triggered in child and handle in parent
  • @Output('aliasName') in ts, (aliasName)="method($event)" in html
More

Data Sharing With Services

  • @Injectable with providedIn: 'root' or providers: [] in app.module.ts
  • Using EventEmitter and emiting events from Service
  • Component A can invoke method which emits the event and pushes data
  • Component B can subscribe to event and access the data emitted in Service
